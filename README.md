# Markit - Enhanced Comments

Make your code **more readable** and **visually organized** with beautifully colored, purpose-driven comments.

---

## ✨ Features

✅ Highlight special comments like:

- `// TODO: fix this`
- `// BUG: handle null cases`
- `// QUESTION: why is this async?`
- `// DEPRECATED: old logic`
- `// HIGHLIGHT: important section`

✅ Works on **any theme** – keeps your current theme and only styles comments.

✅ Supports:

- JavaScript / TypeScript
- C / C++ / C#
- Java / Python / Go / PHP / Ruby
- And more...

---

## 🛠 Supported Tags

| Tag           | Example                      | Color         |
| ------------- | ---------------------------- | ------------- |
| `TODO:`       | `// TODO: Fix the loop`      | Blue-ish      |
| `BUG:`        | `// BUG: Crashes on input`   | Yellow        |
| `ERROR:`      | `// ERROR: Invalid state`    | Red           |
| `HIGHLIGHT:`  | `// HIGHLIGHT: Key logic`    | Green         |
| `DEPRECATED:` | `// DEPRECATED: use newFunc` | Gray (italic) |
| `QUESTION:`   | `// QUESTION: necessary?`    | Purple        |

---
